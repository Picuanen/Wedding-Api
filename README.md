
# Wedding Api

api dari aplikasi wedding


## Installation


Clone Repository 

```bash
  git clone https://github.com/Picuanen/Wedding-Api.git
```
    
Create .env

```bash
  cp .env.example .env
```
Install depedencies with npm

```bash
  cd Wedding-Api/
  npm install 
```
    
Run Project 

```bash
  npm start 
```
